Adifa - Annotated Data in Flask App

Adifa is a framework for visualising single-cell gene expression data in a web browser. It is built on Flask, a micro web framework, and ingests Annotated Data objects in the .h5ad file format. It includes dimensionality reduction visualisation, heatmaps and dotplots, with the ability to explore gene expression and disease markers. The Python- based implementation and usage of the deck.gl framework allows efficient handling of datasets up to one million cells.

Discuss usage on Discourse. Read the documentation. If you’d like to contribute by opening an issue or creating a pull request, please take a look at our contributing guide. If Adifa is useful for your research, consider citing the software.
